# AI Native Research

This folder contains cross-cutting KiSurf AI Native system research.

Use
`results/kisurf-ai-native-system-synthesis-architecture-and-implementation-roadmap.md`
as the current KiSurf AI Native system synthesis, architecture, and
implementation roadmap unless a newer ADR or implementation spec supersedes it.

The master report is intentionally not a context-management report and is not
scoped to any single subsystem. It is the cross-cutting system synthesis
document for product architecture, Chat Agent, Next Action Agent, agent
boundaries, session runtime, layered tools, script execution, provider message
assembly, artifacts, memory and retrieval, multimodal observations, validation,
hidden attempts, preview/review/accept governance, ownership, observability,
implementation status, and sequencing.

Agent-specific research remains in sibling folders:

- `../chat-agent/`
- `../next-action/`
